# Superstore Analytics Project **End-to-end retail analytics using Python, MySQL, and Power BI** 

This project performs a comprehensive analysis of the Superstore dataset (2015–2018) and demonstrates a complete analytics workflow:

* **Python** → data cleaning, feature engineering, forecasting
* **MySQL** → data modeling, ETL, SQL analysis
* **Power BI** → dashboarding and KPI visualization The goal is to replicate a real-world BI/Analytics pipeline and present insights in both technical and business formats.
